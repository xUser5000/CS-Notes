# Explanation
- Volatile memory lose information when powered off.
- It's much faster than [[Non-volatile Memory]].
- Most RAMs are made of volatile memory.
- Types: SRAM and DRAM

- Usage: [[CPU Cache]], main memories, frame buffers (graphics).
